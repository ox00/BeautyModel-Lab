# 数据字典模板（v1）

> 说明：以下为 MVP 阶段建议表。实际项目可按此模板扩展字段。

## 1) product_sku（商品主表）
| 字段 | 类型 | 必填 | 示例 | 说明 |
|---|---|---|---|---|
| sku_id | string | Y | SKU12345 | 主键 |
| brand | string | Y | BrandA | 品牌名 |
| category_l1 | string | Y | 护肤 | 一级类目 |
| category_l2 | string | N | 精华 | 二级类目 |
| price | float | Y | 289.0 | 标价 |
| price_band | string | Y | mid | low/mid/high |
| launch_date | date | N | 2026-01-10 | 上新日期 |
| efficacy_tags | array<string> | Y | ["修护","保湿"] | 功效标签 |
| inci_list | array<string> | Y | ["烟酰胺","神经酰胺"] | 成分表 |
| channel | string | N | xhs/ecom | 来源渠道 |
| is_active | bool | Y | true | 是否在售 |

## 2) ingredient_knowledge（成分知识表）
| 字段 | 类型 | 必填 | 示例 | 说明 |
|---|---|---|---|---|
| ingredient_id | string | Y | ING001 | 主键 |
| ingredient_name | string | Y | 烟酰胺 | 成分名 |
| aliases | array<string> | N | ["维B3"] | 别名 |
| primary_effects | array<string> | Y | ["提亮","控油"] | 主要功效 |
| skin_suitability | array<string> | N | ["油皮","混油"] | 适配肤质 |
| risk_level | string | Y | medium | low/medium/high |
| contraindications | array<string> | N | ["高浓度酸类同用谨慎"] | 搭配禁忌 |
| evidence_level | string | Y | B | A/B/C |
| source_type | string | Y | 法规/词典/期刊/专家 | 来源类型 |
| updated_at | datetime | Y | 2026-02-25 10:00:00 | 更新时间 |

## 3) review_feedback（口碑反馈表）
| 字段 | 类型 | 必填 | 示例 | 说明 |
|---|---|---|---|---|
| review_id | string | Y | RV0001 | 主键 |
| sku_id | string | Y | SKU12345 | 外键 |
| review_date | date | Y | 2026-02-01 | 评论时间 |
| sentiment | int | Y | 1 | -1/0/1 |
| efficacy_mentions | array<string> | N | ["保湿","不搓泥"] | 提及功效 |
| risk_terms | array<string> | N | ["刺痛"] | 风险词 |
| user_skin_bucket | string | N | 油敏肌 | 弱画像桶 |
| source | string | Y | xhs/ecom | 数据源 |
| text_summary | string | N | 上脸清爽... | 摘要 |

## 4) trend_signal（趋势信号表）
| 字段 | 类型 | 必填 | 示例 | 说明 |
|---|---|---|---|---|
| keyword | string | Y | 早C晚A | 趋势词 |
| date | date | Y | 2026-02-24 | 日期 |
| heat_score | float | Y | 78.5 | 热度分 |
| growth_7d | float | Y | 0.42 | 7日增长率 |
| platform | string | Y | xhs/douyin | 平台 |
| related_ingredients | array<string> | N | ["VC","VA"] | 关联成分 |
| related_categories | array<string> | N | ["精华","面霜"] | 关联类目 |

## 5) compliance_rule（合规规则表）
| 字段 | 类型 | 必填 | 示例 | 说明 |
|---|---|---|---|---|
| rule_id | string | Y | C001 | 主键 |
| rule_type | string | Y | 禁用词/敏感词/免责声明 | 规则类型 |
| pattern | string | Y | 治疗痤疮 | 匹配规则 |
| severity | string | Y | high | low/medium/high |
| action | string | Y | block/warn/replace | 处理动作 |
| reference | string | N | 某规范第X条 | 依据 |
| updated_at | datetime | Y | 2026-02-20 12:00:00 | 更新时间 |

## 6) profile_bucket（用户弱画像表，可选）
| 字段 | 类型 | 必填 | 示例 | 说明 |
|---|---|---|---|---|
| profile_id | string | Y | P001 | 主键（匿名桶） |
| skin_type_bucket | string | Y | 油皮/干皮/混油/敏感 | 肤质桶 |
| budget_bucket | string | Y | low/mid/high | 预算桶 |
| concern_bucket | array<string> | Y | ["修护","控油"] | 诉求桶 |
| avoid_ingredients | array<string> | N | ["香精"] | 避免成分 |

## 字段治理要求
- 主键唯一性：100%。
- 时间字段统一：`YYYY-MM-DD` 或 `YYYY-MM-DD HH:MM:SS`。
- 枚举字段必须受控（文档定义可选值）。
- 文本字段默认 UTF-8，禁止写入可识别个人身份信息。
