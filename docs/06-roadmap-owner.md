# 06 - Roadmap and Owners (2+2+1)

## Team model
- 1 x (data + architecture + domain)
- 2 x Engineering (A/B)
- 2 x Product/PM

## 4-week execution sprint
### Week 1 - Freeze
- MVP scope freeze
- data/component/eval contracts freeze

### Week 2 - Integrate
- end-to-end runnable path
- first internal demo

### Week 3 - Validate
- baseline and ablation runs
- failure analysis and fix list

### Week 4 - Iterate and package
- second iteration with error backflow
- mentor-ready proposal + demo artifacts

## Weekly cadence
- Monday: planning (30 min)
- Wednesday: technical sync (30 min)
- Friday: demo + retro (45 min)
